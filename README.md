📌 Descrição

Este repositório contém a implementação de algoritmos em JavaScript com foco no uso de estruturas de repetição para resolução de problemas básicos.

Os programas foram desenvolvidos com o objetivo de praticar lógica de programação, entrada e saída de dados, e versionamento utilizando Git.

🎯 Resultados Esperados
Programas desenvolvidos em JavaScript (.js).
Execução correta após testes.
Utilização de estruturas de repetição (for, while, etc.).
Versionamento dos arquivos em repositório local e remoto utilizando Git.
🛠️ Tecnologias Utilizadas
JavaScript
Node.js (opcional para execução)
Git e GitHub
(Opcional) Flowgorithm para visualização dos algoritmos

📋 Lista de Exercícios
1️⃣ Contagem até o número digitado

Programa que solicita um número ao usuário e exibe todos os números de 1 até ele.

2️⃣ Soma de 10 números

Programa que solicita 10 números inteiros ao usuário e exibe a soma total ao final.

3️⃣ Tabuada

Programa que solicita um número e exibe a tabuada de 1 a 10.

4️⃣ Média de notas

Programa que solicita 5 notas e calcula a média entre elas.

5️⃣ Números pares no intervalo

Programa que solicita dois números e exibe todos os números pares entre eles.

Exemplo:
Entrada: 11 e 20
Saída: 12, 14, 16, 18, 20

▶️ Como Executar
Instale o Node.js (se ainda não tiver)
