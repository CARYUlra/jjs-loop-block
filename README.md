# 📘 JavaScript Loop Exercises

## 📌 Overview

This repository contains a collection of JavaScript programs developed to solve fundamental algorithmic problems using **loop control structures**.

The project emphasizes clean logic implementation, structured problem-solving, and proper version control practices using Git.

---

## 🎯 Objectives

* Apply core programming logic using JavaScript
* Practice **iteration structures** such as `for` and `while`
* Handle user input and output operations
* Organize code into modular files
* Use Git for version control (local and remote)

## 📚 Implemented Solutions

### 1. Counting up to a Number

Prompts the user for a number and prints all values from **1 up to the given number**.

**Key concepts:**

* Iteration with `for`
* Sequential output

---

### 2. Sum of 10 Integers

Collects **10 integer inputs** from the user and calculates their total sum.

**Key concepts:**

* Loop control
* Accumulator pattern

---

### 3. Multiplication Table

Generates the multiplication table (1–10) for a given number.

**Key concepts:**

* Arithmetic operations
* Structured output formatting

---

### 4. Average of Grades

Calculates the average of **5 user-provided grades**.

**Key concepts:**

* Aggregation
* Basic statistics (mean calculation)

---

### 5. Even Numbers in a Range

Displays all **even numbers** between two user-defined values.

**Example:**

```
Input: 11, 20  
Output: 12, 14, 16, 18, 20
```

**Key concepts:**

* Conditional logic (`if`)
* Modulo operator (`%`)
* Range iteration
