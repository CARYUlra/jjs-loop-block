# 📘 Exercícios de JavaScript com Estruturas de Repetição

## 📌 Visão Geral

Este repositório contém uma coleção de programas em JavaScript desenvolvidos para resolver problemas fundamentais de lógica utilizando **estruturas de repetição**.

O projeto foca na implementação de lógica limpa, organização de código e boas práticas de versionamento com Git.

---

## 🎯 Objetivos

* Aplicar lógica de programação com JavaScript
* Praticar **estruturas de repetição** (`for`, `while`)
* Trabalhar com entrada e saída de dados
* Organizar o código em arquivos modulares
* Utilizar Git para versionamento (local e remoto)

---

## 🛠️ Tecnologias Utilizadas

* JavaScript (ES6+)
* Node.js
* Git e GitHub
* (Opcional) Flowgorithm para visualização de algoritmos

---

## 📁 Estrutura do Projeto

```bash
/exercicios-js
│
├── exercicio1.js   # Contagem até o número informado
├── exercicio2.js   # Soma de 10 números inteiros
├── exercicio3.js   # Tabuada
├── exercicio4.js   # Média de 5 notas
├── exercicio5.js   # Números pares em um intervalo
└── README.md
```

---

## 📚 Soluções Implementadas

### 1. Contagem até um Número

Solicita um número ao usuário e exibe todos os valores de **1 até o número informado**.

**Conceitos aplicados:**

* Estrutura de repetição (`for`)
* Saída sequencial de dados

---

### 2. Soma de 10 Números Inteiros

Coleta **10 números inteiros** informados pelo usuário e calcula a soma total.

**Conceitos aplicados:**

* Controle de laço
* Uso de acumulador

---

### 3. Tabuada

Gera a tabuada (de 1 a 10) para um número informado pelo usuário.

**Conceitos aplicados:**

* Operações matemáticas
* Formatação de saída

---

### 4. Média de Notas

Calcula a média de **5 notas** fornecidas pelo usuário.

**Conceitos aplicados:**

* Acumulação de valores
* Cálculo de média aritmética

---

### 5. Números Pares em um Intervalo

Exibe todos os **números pares** entre dois valores informados pelo usuário.

**Exemplo:**

```
Entrada: 11, 20  
Saída: 12, 14, 16, 18, 20
```

**Conceitos aplicados:**

* Estrutura condicional (`if`)
* Operador módulo (`%`)
* Iteração em intervalo

## ▶️ Como Executar

Certifique-se de ter o Node.js instalado. Em seguida:

```bash
git clone https://github.com/seu-usuario/seu-repositorio.git
cd exercicios-js
node exercicio1.js
```

Para executar outro exercício, basta substituir o nome do arquivo.
